Project Overview

The goal of this project was to improve the user flow for users creating an account and ICEBREKR profile. I collaborated with the Nashville Entrepreneur Center and the CEO of ICEBREKR to help revise their onboarding flow and improve the process.

These suggestion presented and implemented orginate from the Nashville Entrepreneur Center UX event, in which local startups reach out to UX researchers and designers for pro bono advice.
